# 🔐 Flutter Authentication App

A simple and clean Flutter application that implements **user authentication** using **Firebase Authentication** and **Cloud Firestore**.  
The app supports Email/Password login, Google Sign-In, user registration, password reset, and a basic user profile screen.

---

## Features

- 🔑 User Sign Up (Email & Password)
- 🔐 User Login
- 🔄 Password Reset via Email
- 🌐 Google Sign-In Integration
- 👤 User Profile Screen (Firestore data)
- 🚪 Logout functionality
- 📦 Firebase Authentication integration
- ☁️ Cloud Firestore for storing user data

---

## Tech Stack

- Flutter
- Firebase Authentication
- Cloud Firestore
- Google Sign-In
- Dart
